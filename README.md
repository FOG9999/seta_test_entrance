# seta_test_entrance
## Frontend
### How to run project
cd frontend
cd seta_posts
npm install
npm start
## Backend
### How to run project
INSTALL DEPENDENCIES
npm install
INSTALL DEVDEPENDENCIES
npm i -D
RESTORE DATABASE
- Open Command Prompt in folder bin of POSTGRES
- Copy the dump.sql file in config folder and paste it in
- Run psql -U [username] [database_name] < dump.sql
START SERVER
npm test
